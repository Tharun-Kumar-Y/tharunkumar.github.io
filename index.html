Projects

Pocket OCR

Description: Pocket OCR is a mobile application designed to capture and convert text from images into editable text using Optical Character Recognition (OCR) technology. The app provides users with a convenient tool to digitize printed documents, business cards, and handwritten notes directly from their smartphones.

What I Did: I developed the Pocket OCR application to enhance text extraction and management capabilities for users. The project involved designing an intuitive user interface, implementing efficient OCR algorithms, and integrating features for text editing and export. I also focused on optimizing the app for various devices and ensuring high accuracy in text recognition.

Technologies Used:

  • Programming Languages: Java (for Android), Swift (for iOS)
  • OCR Library: Tesseract OCR
  • Frameworks: Android SDK, iOS UIKit
  • Backend: Firebase for data storage and authentication
  • Development Tools: Android Studio, Xcode
  • Version Control: Git for source code management

Features:

  • Real-time text extraction from images
  • Support for multiple languages
  • Text editing and formatting options
  • Export to various formats (e.g., PDF, DOCX)
  • Integration with cloud storage services for easy access and sharing

Nom-Nom Foodies

Description: Nom-Nom Foodies is a web application designed to help users discover and share their favorite recipes. The platform allows users to browse a wide variety of recipes, save their favorites, and contribute their own culinary creations. It also features user reviews and ratings to help others find the best recipes.

What I Did: I was responsible for developing the front-end and back-end components of Nom-Nom Foodies. This included designing an engaging user interface, implementing a recipe search and filter system, and setting up a user authentication system. I also integrated a database to manage user data and recipes efficiently.

Technologies Used:

  • Programming Languages: JavaScript, HTML, CSS
  • Front-End Framework: React.js
  • Back-End Framework: Node.js with Express
  • Database: MongoDB
  • Version Control: Git for source code management

Features:

  • Recipe search and filter functionality
  • User registration and login
  • Ability to save favorite recipes and submit new ones
  • Review and rating system for recipes
  • Responsive design for mobile and desktop users
